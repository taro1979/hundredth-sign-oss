# Code Of Conduct

This project uses a simple contributor standard:

- Be respectful and direct.
- Focus criticism on code, behavior, documentation, and operational risk.
- Do not harass, threaten, or disclose private information.
- Do not post exploit details for unresolved vulnerabilities in public issues.

Maintainers may edit, hide, or remove comments and contributions that make the
project unsafe or unproductive for users and contributors.
