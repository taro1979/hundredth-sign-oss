# レビューチェックリスト

## API / Backend
- 入力検証があるか
- 認可が適切か
- エラー処理が適切か
- N+1 や不要な直列処理がないか

## Frontend
- 権限に応じた表示制御があるか
- ローディング、空状態、エラー状態があるか
- 危険な HTML 挿入がないか

## Security
- 秘密情報がコードにないか
- IDOR や権限漏れがないか
- 外部入力を信頼していないか

## Testing
- Happy Path があるか
- 異常系または境界値があるか
- 変更に対応するテストがあるか

## Documentation
- 関連仕様が更新されているか
- TODO やアーキテクチャ文書にズレがないか
